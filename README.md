### miniprogram
基于taro的学习

### 技术
+ taro
+ taro UI
+ fastmock

### 构建
##### 安装依赖，或 npm i
yarn

##### 运行小程序，编译后的文件位于项目下的 dist 文件夹
npm run dev:weapp

##### 运行 H5
npm run dev:h5
